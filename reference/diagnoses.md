# Test dataset of ICD diagnoses

A simulated dataset of hospital diagnoses created with the help of the
`comorbidity` package including the following columns:

- id:

  Individual identifier,

- case:

  Indicator for case status,

- diag:

  An ICD-10 diagnosis code.

## Usage

``` r
data(diagnoses)
```

## Format

A data frame with 23,144 rows and 3 columns
